## Завдання 1
Створіть однотабличну базу даних “Книги”. В ній необхідно зберігати:
* Назва книги
* ПІБ автора
* Рік випуску
* Жанр книги
* Кількість сторінок
* Короткий опис книги

## Завдання 2
Створіть веб-додаток для відображення даних із першого завдання. Використайте Hibernate для доступу
до даних.

## Завдання 6:

* Назва книги
* ПІБ автора
* Рік випуску
* Жанр книги
* Кількість сторінок
